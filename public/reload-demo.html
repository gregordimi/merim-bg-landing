Reload Button Implementation - All Charts

The reload button appears in two scenarios:

1. Error State with Reload Button
When chart fails to load data

Failed to load data

2. No Data Available State with Reload Button
When no data matches the selected filters

No data available for the selected filters

Charts Updated with Reload Functionality:

  • ✅ PieChartComponent
  • ✅ RadarChartComponent
  • ✅ MunicipalityHorizontalChart
  • ✅ SettlementHorizontalChart
  • ✅ MultiLineTrendChart

How it Works:

  1. Each chart now has a refreshKey state
  2. When reload button is clicked, handleReload() increments the key
  3. This triggers a new data query through useStableQuery
  4. Fresh data is fetched from the API
